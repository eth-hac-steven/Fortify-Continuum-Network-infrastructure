# Threatsphere-Consulting-Network-infrastructure
A comprehensive Threatsphere Consulting Network infrastructure lab featuring a hardened pfSense firewall, Windows Server 2022 Active Directory domain controller, two joined client workstations, and a NAS for centralized storage. Designed for simulating enterprise Help desk and security scenarios, penetration and Defensive testing, and network segmentation strategies and more.

## Prerequisites
- VirtualBox ≥ 7.x installed on host (Windows/macOS/Linux)
- ISO images (download latest versions):
  - pfSense-CE-2.8.x-amd64.iso
  - win server 2022.iso
  - win 11.iso
  - True-NAS.iso
  - kali-linux-current-amd64.iso
  - ubuntu-22.04-desktop-amd64.iso
- Minimum host resources: 16 GB RAM, 140 GB free disk, def use more 😂
